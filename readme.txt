== Description ==

Display 360° photos and videos on your blog using WP Panellum Panoramic VR free API simply by using a shortcode. 

To know more about Panellum visit the official website at https://pannellum.org

== Installation ==
1 - download plugin
2 - Activate Plugin
3 - Use the shortcode [panoramicvr src="http://www.domain.com/yourimage.jpg"]

Parameters: width, height


== Frequently Asked Questions ==
= How to change the size ? =
Use shortcode parameters [panoramicvr width="XXX" height="XXX" src="http://www.domain.com/yourimage.jpg"]. 

The size can be define in pixel or percents
